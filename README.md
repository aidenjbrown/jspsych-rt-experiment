# Behavioral task (reaction-time + copy-typing)

A short (~3 min) browser experiment for research data collection. Open the live
page, complete the task naturally, and use the **Download JSON** button on the
final screen to save your data. Send the downloaded file to the researcher.

Live page: https://aidenjbrown.github.io/jspsych-rt-experiment/

The page records interaction events (mouse, keyboard, timing) locally in your
browser using the jsPsych `record_session` schema; nothing is uploaded — you
choose to download and share the file.
